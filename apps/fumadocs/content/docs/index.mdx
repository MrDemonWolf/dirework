Dirework is a self-hosted Pomodoro timer and task list designed for Twitch co-working and body-doubling streams. Streamers login with Twitch, connect a bot account, configure OBS overlays, and their viewers can interact via chat commands.

## Features

- Pomodoro timer with configurable work/break durations
- Task list where viewers can add and manage tasks via chat
- OBS browser source overlays for timer and task list
- Twitch bot integration for chat commands
- Fully customizable overlay styling
- Single-user per instance (self-hosted)
